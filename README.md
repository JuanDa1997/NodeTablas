# Notas:
Programa para multiplicar en Node
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar      [número] [requerido]
  -l, --listar   Este es el numero hasto donde quieres multiplicar
                                                     [booleano] [defecto: false]
  -h, --hasta    Muestra la tabla en pantalla           [número] [defecto: "10"]
```
